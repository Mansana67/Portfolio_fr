Roban'Adala

Résumé

"Roban'adala" ou "le braquage fou" est une animation 2D centrée autour de quatre personnages : Collette la conductrice, Doda le macho, Rainboto le débile et Vazo le chanteur. Ensemble, ils s'entraident pour braquer une banque malgré leur manque de compétences.
Lien de l'animation : https://youtu.be/GfBKd3JIm1U

Popotes Store

Résumé

"Popotes Store", une animation combinée 3D et 2D, est une publicité visant à promouvoir le site Web Popotes Store afin d'aider les gens pendant la pandémie à acheter leurs besoins en ligne.
Lien de l'animation : https://youtu.be/6eL8yZsnBbU

  JodaRak

Résumé

JodaRak est une animation 3D créée pour montrer la transformation du design de voiture, passant de croquis 2D à une modélisation 3D complexe, pour se terminer par une courte animation servant de publicité pour le design.
Lien de l'animation : https://youtu.be/E2IrIb7E78U
